
Docker 是一个开源的容器化平台，用于将应用程序及其依赖打包到轻量级容器中，实现快速部署与跨环境一致性。Docker 使用镜像机制来构建应用环境，支持通过 Dockerfile 定义镜像内容。 主要功能包括容器运行、镜像管理、网络配置、数据卷挂载、Docker Compose 多容器编排等。适用于开发、测试、生产环境中的微服务与 DevOps 实践。
Postman 是一款流行的 API 开发与测试工具，支持构造和发送 HTTP 请求（如 GET、POST），并查看响应结果。它提供可视化的请求构造界面，支持 JSON、XML 等数据格式。 主要功能包括 API 调试、自动化测试脚本、Mock 服务、API 文档生成、团队协作与版本管理。Postman 适用于 RESTful API 和 GraphQL 接口的开发与测试。
